# Holol POS Cafe LM — Windows Station

Téléchargement officiel public de l’installateur Windows pour Cafe LM.

## Installation rapide

1. Téléchargez `HololPOS-CafeLM-Setup-1.2.0.exe` depuis la dernière release.
2. Double-cliquez sur le fichier.
3. Cliquez `Suivant` → `Installer` → `Terminer`.
4. Laissez cochée l’option pour lancer Holol POS.
5. Au premier lancement, Holol détecte l’imprimante Windows, lance un test, puis affiche `Prêt`.

Si Windows SmartScreen apparaît : cliquez `Informations complémentaires`, puis `Exécuter quand même`.

## Fichiers publiés

- `HololPOS-CafeLM-Setup-1.2.0.exe` — installateur Windows principal.
- `SHA256SUMS.txt` — empreinte SHA256 de vérification.

Le code source de Holol POS reste privé dans `povssam/7olol-pos`.
